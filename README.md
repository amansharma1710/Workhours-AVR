# Workhours-AVR
Workhours
          ˃Simple effective way to calculate work hours of all employees of a small organisation˂

1. The workbooks simply makes it easy for an organisation to keep records of the workhours of the employees for payment. 

2. The entries can be done in a centralised manner or by the employees themselves.

3. Entries will include employees name, time of the start of an employee's shift, and the time of its end. 

4. The names are to be entered on the first day of the month, the worksheets of the later days will display those names in the same order. 

5. Hence, Each day is a new Worksheet, and each month is a new workbook. 

6. The whole file is to be uploaded on a cloud so the changes made to the files can be tracked and should be given access to at least one other account as 'author' other than the uploader. The Sheets' access can be given to all the employees with an account to the cloud platform(on which the employer decides to upload the file on) once a system to review changes is established. 

7. This is to be done so the hours done can't be altered in unsupervised manner. 

8. System can only work on the trust of the employees and employer/s.
